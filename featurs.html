Lyre Discord Bot - Feature List

Overview

Lyre is a feature-rich Discord bot created by Lyre Development, designed for server management with a focus on ticket systems, monitoring, and administration.

Core Features

1. Ticket System

  • Creates and manages support tickets
  • Ticket categories: General, Reward, Purchase, and Report
  • Ticket actions: Claim, Close, Alert
  • Automatic ticket transcripts
  • Ticket activity tracking
  • Ticket inactivity monitoring

2. Command System

  • Uses Discord slash commands for easy interaction
  • Command permissions management via Discord's permission system
  • Automatic command registration with Discord API

3. Invite Tracking

  • Monitors who invited users to the server
  • Tracks invite usage statistics

4. User Leveling System

  • Tracks user activity and assigns levels
  • Stores user progression data

5. Web Dashboard

  • Web interface for managing bot settings
  • Remote monitoring capabilities

Commands List

General Commands

  • /ping - Checks the bot's latency and uptime (Anyone can use)

Giveaway Commands

  • /gend - Ends a giveaway (Requires admin/moderator permissions)

Ticket Management

  • Various ticket-related commands for creating, managing, and closing tickets
  • Most ticket commands require specific staff roles

Moderation Commands

  • Commands for managing users and server content (Requires moderator/admin permissions)

Permission System

The bot uses Discord's built-in permission system:

  • Commands have permission requirements defined in their construction
  • Example: Admin-only commands use PermissionFlagsBits.Administrator
  • Moderator commands typically require PermissionFlagsBits.ManageMessages or similar
  • Ticket management often requires specific roles that can be configured

Technical Features

  • Written in Node.js using Discord.js library
  • Real-time status monitoring and dynamic presence updates
  • Hot reloading of commands in development mode
  • Comprehensive logging system
  • Uses embedded database for ticket and user information
  • Custom emoji support for rich visual feedback

Setup Requirements

  • Discord Bot Token
  • Server with Node.js
  • Appropriate bot permissions in Discord
